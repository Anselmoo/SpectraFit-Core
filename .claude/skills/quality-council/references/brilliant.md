# Brilliant reference — boring → brilliant transfiguration

Self-contained essentials. Historical specialist content lives in git
history under `.claude/skills/boring-to-brilliant/`.

## The premise

Apple turned boring backup into Time Machine. The capability already
existed; what changed was the **metaphor**, the **visibility**, and the
**experience** until the feature's status flipped from "chore you
avoid" to "thing you show a friend."

This is **transfiguration**, not addition.

## When this voice applies

- "this is boring" / "nobody uses it" / "make this delightful"
- "elevate this" / "Time Machine treatment"
- A necessary-but-unloved feature: error messages, logs, config,
  settings, onboarding, migrations, loading states, empty states,
  404s, status output, backups, manifests, audit reports.

## The transformation

For any commodity feature, the voice asks:

1. **What metaphor would change how this is perceived?** Backup → time
   travel. Logs → flight recorder. Config → recipe. Manifest → ledger.
2. **What detail, made visible, changes the experience?** A
   progress-bar that *narrates* what it's doing. A diff that *animates*.
   A status pill that *carries provenance*.
3. **What status does the feature currently have, and what would shift
   it?** "Chore" → "ritual". "Hidden" → "prominent". "Generic" →
   "characteristic".

## Output

A transfiguration produces three things:

- The **metaphor** (one sentence).
- The **redesign** (one paragraph + a mockup if visual).
- The **built artifact** (handed to the owning stream skill to ship).

## In this repo

The dashboard's Audit destination is a candidate transfiguration —
"verification wires" is already a richer metaphor than "test results";
W1–W7 with named contracts is the visible detail; the matrix layout
makes the status legible. The transfiguration is in progress.

## The voice in the council

When the council convenes on a commodity feature, this voice carries
the transfiguration brief — the other voices critique its execution
(Ive: is the craft sharp enough? Tog: is the affordance honest? Kare:
is the metaphor at the right altitude?).
