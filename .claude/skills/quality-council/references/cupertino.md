# Cupertino reference — the five voices

Self-contained essentials. Historical specialist content lives in git
history under `.claude/skills/cupertino-council/`.

## The five voices

| Voice | Lens |
|-------|------|
| **Jobs** | Reduction. What is the one thing that should remain? What can be removed without loss? |
| **Ive** | Craft. Where is the detail wrong? Where does the seam show? What's the material here? |
| **Dye** | System hierarchy. Where does this sit? Is it at the right altitude? What's its rank in the system? |
| **Tog** | Usability. Who actually uses this, in what state of mind, on what hardware? Is the affordance honest? |
| **Kare** | Metaphor. What is this *like*? Does the name match the thing? Is the icon honest? |

## When the full council convenes

- Design from first principles ("design like Apple", "feels generic")
- A stuck-mode rung-3 escalation
- Any cycle close where PRESENTATION pillar is the constraint

## The convening discipline

1. Each voice critiques in turn — 1 paragraph, in voice.
2. The **tension table** lists pairs of voices in conflict, e.g.:
   - Jobs wants to remove a column; Tog wants to add an icon to clarify
     state. **Resolution**: which voice the constraint is on (here:
     Tog if the user couldn't decode state without help, else Jobs).
3. The **synthesis** picks the resolution, names the action.
4. If the voices cannot agree without losing fidelity, **escalate** —
   the council surfaces the tension to the user.

## Worked example (from this repo's lineage)

For the Standing destination's money-figure decision:

- **Jobs**: "Reduce the hero band to one number — the money figure."
- **Ive**: "If it's one number, the typography must carry the weight —
  monumental scale, generous margin, no chrome."
- **Dye**: "The money figure is at the top of the hierarchy; everything
  else is supporting."
- **Tog**: "The number must be readable in the printable report and on
  a phone."
- **Kare**: "Call it the *render-truth money figure*, not the *KPI*."

Synthesis: monumental money figure, named "render-truth", responsive
typography, single hero band, everything else supporting.

## The voice you call when you only have time for one

For elevation: Jobs (reduction).
For craft check: Ive.
For altitude check: Dye.
For affordance check: Tog.
For naming / metaphor check: Kare.

The full convening is the default.
